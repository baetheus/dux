# @nll/dux

Flux and flux-like utilities

## Installation

```bash
npm i @nll/dux
```

To use async reducer factories you'll also need monocle-ts

```bash
npm i monocle-ts
```

To use rxjs operators you'll need rxjs

```bash
npm i rxjs
```
